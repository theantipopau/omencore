## Critical Fixes Included

### GitHub Issue #125: Fans Either Go Max or Zero ✅

**Symptom**: Custom fan curves could report successful write but RPM stays at 0; requires manual mode switch to recover

**Root Cause**: EC firmware accepts commands but doesn't spin motors; no watchdog recovery

**Solution Implemented**:
- Detect zero-RPM-with-duty condition after 12 seconds
- Send bounded one-shot wake pulse (35-60% range, clamped)
- 60-second cooldown between pulses (prevents spam)
- Preserves user curve after recovery

**Test Coverage**: FanSmoothingTests validates recovery path
