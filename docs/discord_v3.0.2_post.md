# OmenCore v3.0.2 is live 🔧

Quick stability hotfix release focused on OMEN/Victus telemetry + fan reliability.

## ✅ Fixed in 3.0.2
- CPU temp stuck/frozen behavior mitigation (including persistent low/stale readings)
- Fan preset re-apply after sleep/resume
- OGH telemetry fallback so OMEN tab no longer shows false `0 RPM / 0%` on affected systems
- PawnIO post-install reboot guidance for 0W CPU power edge cases

## 🐧 Linux
Linux x64 package is now built for v3.0.2 and includes both:
- `omencore-cli`
- `omencore-gui`
