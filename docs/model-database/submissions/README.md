# Submissions

Community model-capability submissions land here, one JSON file per board, named `<productid>.json`
(lowercase, e.g. `8d87.json`). See `../CONTRIBUTING_MODEL_DATABASE.md` for the process and
`../model-capabilities.schema.json` for the format.

Files here are the auditable source record for entries merged into
`src/OmenCoreApp/Hardware/ModelCapabilityDatabase.cs` — they stay even after merge so a maintainer (or
anyone else) can trace an entry back to who reported it and what evidence it was based on.
