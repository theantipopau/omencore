## OmenCore v3.2.1 is Live

Download: https://github.com/theantipopau/omencore/releases/tag/v3.2.1

---

### What is in this release

- 23 hotfixes merged since v3.2.0
- Stability and telemetry hardening across temperature, fan, and power paths
- Premium UI polish pass for Hotkey OSD and key control surfaces
- Improved portable-mode diagnostics and cleaner fallback logging
- CPU temperature oscillation guard for affected OMEN 17 sessions

---

### Key highlights

- CPU overheat false-positive protection improved with persistence and sanity checks
- Sleep/resume monitoring recovery strengthened
- Auto-update feed logic improved for prerelease and stable channels
- Fan control safety and reliability improvements (curve behavior and reset behavior)
- Linux docs and diagnostics workflow refreshed
