# OmenCore v3.0.1 🎉 Available Now!

**Stability & Features Update**

We've patched 10 bugs from community reports and added 4 major UX enhancements to the Memory Optimizer and Bloatware Manager.

## ✅ What's Fixed
- **Temperature Readings** — Intel Core Ultra / Arrow Lake now report correct CPU temps
- **Fan Control Stability** — Removed glitches that caused false RPM spikes
- **GPU Temperatures** — Fixed freezing-at-idle issues with NVIDIA sensors
- **Power Monitoring** — CPU and GPU power now read correctly (no more 0W)
- **Quick Profiles** — Performance mode selection now syncs across all tabs
- **Keyboard Shortcuts** — Function key conflicts with system shortcuts resolved
- **Game Library** — Launch/edit buttons stay responsive after game selection
- ...and 3 more fixes for bloatware scanning, data corruption, and EC safety

## ✨ What's New
- **Memory Profiles** — Conservative/Balanced/Aggressive presets with live cleanup preview
- **Process Ranking** — See your top 10 RAM hogs updated every 2 seconds
- **Cleanup Preview** — Estimate how much memory will be freed before you clean
- **Bulk Restore** — Restore all removed bloatware in one click
